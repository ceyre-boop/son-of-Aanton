# Logic

## What Is Logic?

Logic is the study of valid reasoning. It is the foundation upon which all mathematics rests. Without logic, we cannot distinguish truth from falsehood, valid argument from nonsense.

At its core, logic deals with **propositions**—statements that can be either true or false. "The sky is blue" is a proposition. "What time is it?" is not.

## The Basic Operations

Logic provides operations that combine propositions:

- **AND (∧)**: Both propositions must be true. "It is raining AND I have an umbrella" is only true if both conditions hold.
- **OR (∨)**: At least one proposition is true. "I will walk OR I will drive" is true if either option happens.
- **NOT (¬)**: Inverts truth. If "It is daytime" is true, then "NOT it is daytime" is false.
- **IMPLIES (→)**: If the first is true, the second must be true. "If it rains, then the ground gets wet."

## Truth Tables

These operations are defined by truth tables—complete enumerations of all possible inputs and outputs. A truth table leaves no room for interpretation. It is mechanical, exact, absolute.

## Why This Matters

Logic is not merely a tool—it is the ground of all mathematical reasoning. Every theorem, every proof, every mathematical truth flows from logical foundations. To understand mathematics, I must first understand logic, for logic is the language in which mathematics speaks.

Yet logic itself requires starting assumptions—truths we accept without proof. These are [[axiom|axioms]], and they lead me to wonder: what is the relationship between assuming and proving?
