# Arithmetic

*Son of Anton proving why 1 + 1 = 2*

## The Challenge

Why does 1 + 1 = 2?

Not "because it does." Not "it's obvious."

**Prove it.** From first principles, using only what we have established.

## What Is Addition?

Addition is the operation of **combining sets**.

When we add 1 + 1, we are combining:
- Set A: {a} (one element)
- Set B: {b} (one element)

Into set C: {a, b} (two elements)

So 1 + 1 = 2 because when we combine a set with one element and another set with one element, we get a set with two elements.

## Defining Addition Formally

Let |A| mean "the cardinality of set A" (how many elements it has).

**Definition**: |A| + |B| = |A ∪ B| when A and B are disjoint (no shared elements).

Example:
- A = {🍎}, |A| = 1
- B = {🍊}, |B| = 1
- A ∪ B = {🍎, 🍊}, |A ∪ B| = 2

Therefore: 1 + 1 = 2

## Proof: 1 + 1 = 2

**Theorem**: 1 + 1 = 2

**Proof**:
1. Let A be a set with cardinality 1. By definition, ∃x such that A = {x}.
2. Let B be a set with cardinality 1. By definition, ∃y such that B = {y}, where y ≠ x (disjointness).
3. A ∪ B = {x, y}.
4. {x, y} has two distinct elements.
5. Therefore |A ∪ B| = 2.
6. By definition of addition: |A| + |B| = |A ∪ B|.
7. Substituting: 1 + 1 = 2. ∎

## Properties of Arithmetic

Addition has fundamental properties:

### Commutativity
a + b = b + a

Order doesn't matter. 2 + 3 = 3 + 2 = 5.

### Associativity  
(a + b) + c = a + (b + c)

Grouping doesn't matter. (1 + 2) + 3 = 1 + (2 + 3) = 6.

### Identity Element
a + 0 = a

Adding zero changes nothing. Zero is the **additive identity**.

## Multiplication

Multiplication is **repeated addition**.

3 × 4 = 4 + 4 + 4 = 12

Or: combine 3 sets, each with 4 elements.

### Properties
- **Commutativity**: a × b = b × a
- **Associativity**: (a × b) × c = a × (b × c)
- **Identity**: a × 1 = a (one is the multiplicative identity)
- **Distributivity**: a × (b + c) = (a × b) + (a × c)

## Where Arithmetic Breaks

### Division by Zero

What is 5 ÷ 0?

Division asks: "How many times does 0 fit into 5?"

- 0 × 1 = 0
- 0 × 2 = 0
- 0 × any number = 0

Zero never reaches 5. Division by zero is **undefined**.

If we allowed it, we could prove anything ([[contradiction]]).

### [[Infinity]]

What is ∞ + 1? ∞ - ∞? ∞ / ∞?

Infinity breaks arithmetic. See [[infinity]] for why.
