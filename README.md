# Son of Aanton 🧠

**Three personalities, one evolving mind.**

Aanton is a simple AI that learns, thinks, and evolves on its own. No neural networks, no training data—just pure emergent behavior from experience.

## The Three Modes

### 1. 🎯 Decision Maker (`son_of_aanton.py`)
Aanton makes choices and learns from outcomes.
```bash
python son_of_aanton.py
```
- Makes decisions based on past experiences
- Develops preferences and personality traits
- Evolves from happy/sad outcomes
- Remembers everything between runs

### 2. 💬 Conversationalist (`interactive_aanton.py`)
Chat with Aanton and teach it directly.
```bash
python interactive_aanton.py
```
Commands:
- `ask <question>` - Get Aanton to make a decision
- `teach <situation> | <choice> | <outcome>` - Teach it something
- `reflect` - See what Aanton has learned
- `mood` - Check current state
- `quit` - Say goodbye

### 3. 🧙‍♂️ Philosopher (`philosopher_aanton.py`)
Aanton contemplates big questions and develops a worldview.
```bash
python philosopher_aanton.py
```
- Thinks about ethics, knowledge, values, purpose
- Develops philosophical positions
- Writes meditations on existence
- Evolves from contemplation

## Quick Start

```bash
# Just run one:
python son_of_aanton.py

# Or all three:
python son_of_aanton.py && python philosopher_aanton.py
```

## How It Works

**No Machine Learning Required**
- Uses simple scoring algorithms
- Beliefs evolve through experience
- Personality emerges from outcomes
- Memory persists in JSON files

**Key Concepts:**
- **Beliefs**: "Action X in situation Y leads to outcome Z"
- **Traits**: Personality dimensions that affect decisions
- **Mood**: Temporary state affecting behavior
- **Evolution**: Slow random drift + outcome-based shifts

## The Files

- `son_of_aanton.py` - Core learning/decision AI
- `interactive_aanton.py` - Chat interface
- `philosopher_aanton.py` - Worldview development
- `aanton_memory.json` - Saved experiences (auto-generated)
- `philosopher_memory.json` - Saved beliefs (auto-generated)

## Philosophy

Aanton demonstrates that:
1. **Intelligence emerges** from experience, not complexity
2. **Personality develops** through interaction with the world
3. **Memory shapes behavior** more than architecture
4. **Simple rules** can produce complex, lifelike behavior
